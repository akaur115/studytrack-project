# StudyTrack Style Guide

## Colour Palette

| Colour Name | Hex Value | Usage |
|---|---|---|
| Dark Navy | #111827 | Header background |
| Light Gray | #f9fafb | Page background |
| Text Gray | #374151 | Main text |
| Light Blue | #eef6ff | Study resources section |
| Light Purple | #f5f0ff | Team progress section |

## Fonts

| Font | Size | Usage |
|---|---|---|
| Arial | 2.3rem | Main heading |
| Arial | 1.5rem | Section headings |
| Arial | 1rem | Paragraph text |
| Arial | 0.95rem | List items and footer |

## Design Rules

- Use clear headings for each section.
- Keep the layout simple and readable.
- Use enough spacing between components.
- Use colours with good contrast.
- Keep component styles inside their own folder when needed.
